🎯 INSTRUÇÕES RÁPIDAS (3 PASSOS - 10 MINUTOS):

PASSO 1️⃣  → Criar repositório no GitHub
  • Abra: https://github.com/new
  • Nome: sua-empresa
  • Visibilidade: Public ⭕
  • Clique: Create

PASSO 2️⃣  → Fazer push do PC para GitHub
  • Abra: PowerShell
  • Digite:

    cd "C:\Users\Tatiana\Desktop\Claude Code\FLUMAP\Template - Copia"
    git remote add origin https://github.com/SEU_USUARIO/sua-empresa.git
    git branch -M main
    git push -u origin main

  • Substitua SEU_USUARIO pelo seu nome de usuário do GitHub

PASSO 3️⃣  → Conectar Vercel e fazer deploy
  • Abra: https://vercel.com
  • Login com: Google
  • New Project
  • Selecione o repositório: sua-empresa
  • Deploy (tudo padrão)

  ✅ PRONTO! Seu site estará em:
     https://sua-empresa.vercel.app

❓ DÚVIDAS?

  • Erro ao fazer git push?
    → Pode ser autenticação. Use token pessoal:
      https://github.com/settings/tokens

  • Vercel não encontra repositório?
    → Vá em Settings > Configure GitHub App
    → Autorize Vercel a acessar seus repos

  • Build failed?
    → Verifique logs no Vercel dashboard
    → Geralmente é arquivo com erro de sintaxe
